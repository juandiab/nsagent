    Welcome to JPilot

    This wizard configures everything needed to run the platform: an admin account, your domain, and TLS. Secrets are generated for you. It takes under a minute.

    • Create the admin login
    • Pick the domain JPilot will be served from
    • Choose production or development mode
    • Use your own TLS certificate or generate a self-signed one

    Administrator account

    The first account that can sign in to JPilot.

    Domain & branding

    The hostname users will open in the browser. CORS, the API URL and passkey (WebAuthn) settings are all derived from this automatically.

    JPilot will be reachable at https://localhost. Passkeys are enrolled later from inside the app.

    Deploy mode

    Production builds the UI and runs without hot reload — faster and recommended for real installs. Development keeps source mounted and reloads on change, which is useful when hacking on JPilot locally.

    TLS certificate

    nginx terminates HTTPS. Provide your own certificate, or let the installer generate a self-signed one to get going.
